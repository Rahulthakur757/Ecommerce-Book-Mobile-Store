# RD Project Book Store (MERN Full Stack Project)

This is a full stack MERN (MongoDB, Express, React, Node.js) e-commerce project for selling books and mobile phones.

---

## Project Structure

/backend  
/frontend-user  
/frontend-admin  


## Features

### User Side
- View Books and Mobiles
- Search Products
- Add to Cart
- Buy Products using Stripe Payment
- View Order History
- Add Reviews and Ratings

### Admin Side
- Add, Edit, Delete Books
- Add, Edit, Delete Mobiles
- Manage Discounts
- Manage Delivery Locations
- View Transactions
- Manage Reviews


## Tech Stack

Frontend:
- React.js (Vite)
- Redux Toolkit
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- Cloudinary (Image Upload)
- Stripe (Payment)
- JWT Authentication
- Nodemailer


## How to Run Frontend
/frontend-user

npm run dev

/frontend-admin
